# Instagram Audience Behavior Analysis using Core Python

## Problem Statement
Raw Instagram user data is often unstructured and difficult to analyze directly.  
This project focuses on transforming raw Instagram data into a readable and structured format and performing descriptive analysis to understand audience characteri

## Dataset
The dataset contains raw Instagram user information including:
- Number of posts
- Number of followers
- Number of accounts followed
- User category

## Data Cleaning & Transformation
- Converted raw data into structured, readable formats
- Extracted relevant attributes for analysis
- Prepared cleaned data for further analytical use


## Analysis Performed
- Identified users with the maximum number of posts
- Identified users with the maximum number of followers
- Identified users with the maximum number of following
- Analyzed distribution of users across different categories

## Tools Used
- Python (Core Python)
- Built-in data structures and logic

## Conclusion
This project demonstrates the use of core Python for data preprocessing and descriptive analysis, forming a foundation for more advanced data analysis using libraries such as pandas and visualization tools.
